# 🎨 Tarea Clase 4: Dándole Vida a nuestra Web con CSS 🌟

¡Increíble progreso! Ya pasamos de la estructura pura (HTML) a los estilos y la estética (CSS). Ahora es el momento de que tu página deje de verse como un documento de texto y empiece a verse como un sitio real.

Esta es una **actividad no obligatoria**, pensada para que practiques y experimentes sin miedo a romper nada. ¡El CSS es pura creatividad!

---

## 🎯 El Objetivo
Tu misión es tomar el proyecto que hiciste en la **Clase 3** (tu web personal o el registro de evento) y transformarlo usando CSS. Si quieres empezar algo nuevo desde cero, ¡también eres bienvenido!

---

## 🛠️ Requerimientos Técnicos

Para completar este desafío, tu hoja de estilos (`style.css`) debe incluir al menos:

1.  **Vínculo Externo:** El CSS debe estar en un archivo separado y vinculado correctamente en el `<head>` de tu HTML.
2.  **Paleta de Colores:** Cambia el color de fondo del `body` utilizando `background-color` y el color de los textos principales (`h1`, `h2`, `p`). ¡Busca armonía!
3.  **Tipografía:** Cambia la fuente de todo el sitio usando `font-family`.
4.  **Uso de Selectores:**
    *   Usa al menos un selector de **etiqueta** (ej: `h1 { ... }`).
    *   Usa selectores de **clase** para elementos específicos (ej: `.mi-boton { ... }`).
    *   Usa un selector de **ID** para un elemento único.
5.  **Interactividad (Pseudo-clases):**
    *   Añade un efecto `:hover` a tus enlaces o botones (que cambie el color o el tamaño).
    *   Añade un estilo `:focus` a los campos de tu formulario para que el usuario sepa dónde está escribiendo.
6.  **Pintando Listas o Tablas (`nth-child`):**
    *   Si tienes una tabla o una lista, usa `:nth-child(odd)` o `:nth-child(even)` para pintar filas o ítems de distintos colores (estilo "cebra").
7.  **Detalles Visuales (Pseudo-elementos):**
    *   Usa `::placeholder` para cambiar el color del texto de ayuda en tus inputs.
    *   (Opcional) ¡Anímate a usar `::before` o `::after` para agregar un emoji o un icono antes de tus títulos!

---

## 🚀 Desafío de Especificidad (Nivel Pro)
Intenta crear una regla general para todos tus párrafos `<p>`, y luego crea una regla mucho más específica (usando combinación de clases o IDs) para cambiarle el color a un párrafo en particular. ¡Fíjate cuál gana la batalla de la especificidad!

---

## 💡 Recomendaciones:
*   **Colores:** Puedes usar herramientas como [Adobe Color](https://color.adobe.com/) o [Coolors](https://coolors.co/) para encontrar paletas lindas.
*   **Fuentes:** Recuerda que puedes usar nombres de fuentes genéricas como `sans-serif`, `serif` o `monospace`.
*   **Inspector de Chrome:** Si algo no se ve como quieres, haz clic derecho -> **Inspeccionar**. Ahí podrás ver qué reglas de CSS están afectando a cada elemento y por qué.
*   **¡Diviértete!:** Prueba colores locos, bordes gigantes y filtros. Estamos en la etapa de descubrimiento.
