# 📝 Actividad de Tarea: Flexbox Challenge

## 📋 Objetivo
Practicar el uso de Flexbox para crear componentes comunes de una interfaz web, como una barra de navegación y una sección de tarjetas (cards).

---

## 🛠️ Instrucciones

1.  Crea una nueva carpeta llamada `tarea_flex` dentro de tu proyecto.
2.  Crea un archivo `index.html` y un archivo `style.css`.
3.  Utiliza los conceptos aprendidos en clase para resolver los siguientes dos desafíos:

### Desafío 1: Barra de Navegación (Navbar)
Debes crear un encabezado (`<header>`) que contenga:
*   Un **Logo** a la izquierda.
*   Una **Lista de enlaces** (Inicio, Productos, Contacto) a la derecha.
*   **Requisito:** Usa `justify-content` para separar el logo de los enlaces.

### Desafío 2: Sección de Tarjetas (Cards)
Debes crear un contenedor que tenga **3 tarjetas** iguales:
*   Cada tarjeta debe tener un color de fondo, un borde y un texto.
*   **Requisito:** Las tarjetas deben estar centradas en la pantalla y tener un espacio (`gap`) de 20px entre ellas.
*   **Plus:** Si achicás la pantalla, las tarjetas deben "saltar" abajo una de otra automáticamente (usá `flex-wrap`).

---

## 💡 Tips de Ayuda
*   Recordá siempre poner `display: flex;` en el contenedor padre.
*   Usá `box-sizing: border-box;` en el selector universal (`*`) para no tener problemas con el tamaño de las cajas.
*   Para centrar todo verticalmente en la pantalla, podés darle al contenedor un `height: 100vh;` y usar `align-items: center;`.
